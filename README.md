# SH1106_GUI_Design<br>
SH1106 GUI設計工具<br>

2025/10/15<br>
有鑑於SH1106設計工具複雜，而且可以順便練習Qt的程式語法和設計風格，所以寫了這個程式。<br>
簡單來說就是一個GUI的設計程式,目前的功能只能做到讀檔、轉檔,在屏幕一個一個點出想要的圖片；然後存檔或是轉換出來。<br>
我是不知道演算法對不對，後續會測試。<br>
另外,本程式不支援Arduino<br>
我討厭那東西<br>
